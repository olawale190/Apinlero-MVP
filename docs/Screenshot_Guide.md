## How to Take Screenshots

### On Mac:
- **Full screen:** Cmd + Shift + 3
- **Selection:** Cmd + Shift + 4
- **Window:** Cmd + Shift + 4, then Space, then click window

### Mobile Simulation:
1. Open Chrome/Safari
2. Press F12 (or Cmd + Option + I)
3. Click the mobile/tablet icon
4. Select "iPhone 12 Pro" or similar
5. Take screenshot

## Tips for Good Evidence Screenshots

- Show the full browser URL bar (proves it's live)
- Include the date/time if possible
- Capture on both desktop and mobile
- Show real data (orders, products) not empty states
- Highlight AI features and innovation aspects
